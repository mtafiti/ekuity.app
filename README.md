# Ekuity.app - Mobile
This app that helps users track their favorite equities. 
This is not a real time watchlist. You will need to get your own API keys. They're all free. To get started, [follow these instructions.](https://github.com/JoshuaR503/Stock-Market-App/blob/master/lib/keys/api_keys.dart) [Download here.](https://play.google.com/store/apps/details?id=com.stonks.watchlist)
